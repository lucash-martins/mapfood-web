# MapFood

Este repositório contém a aplicação front-end do projeto MapFood.

Como rodar localmente:

```powershell
git clone <SEU_REPOSITORIO>
cd <SEU_PROJETO>
npm install
npm run dev
```

Tecnologias usadas:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

Para deploy, use o provedor de sua preferência (Vercel, Netlify, Cloudflare Pages etc.).
